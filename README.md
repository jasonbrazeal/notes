# notes

```bash
# clean webpack build dir
npm run clean
# run webpack build
npm run build
# start webpack static server
npm run start
npm run --prefix /path/to/project/folder start
# start python static server (run build just before)
cd build
python -m http.server
# start flask server
NOTES_DIR=/path/to/notes/ FLASK_DEBUG=1 FLASK_APP=/path/to/project/folder/project/app.py flask start
```
